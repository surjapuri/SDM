Enable AI second opinion?

    
Offline and free threat-intel checks always run automatically and never leave your device or this browser's network requests to public lists. An AI second opinion is deeper, but sends the text or image you're checking to our server, which forwards it to Mesh API for analysis. Nothing is stored afterward.
